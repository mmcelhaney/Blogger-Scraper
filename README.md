# Blogger RSS Feed Scraper

A React-based web application to scrape and export your Blogger blog posts into organized XML files by year.

## Features

- Parse Blogger URLs to RSS feeds
- Fetch all blog posts efficiently
- Organize posts by year
- Export to XML format
- Real-time progress tracking
- Responsive UI

## Live Demo

[View Live Demo](https://YOUR-USERNAME.github.io/blogger-scraper)

## Local Development

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000

## How to Use

1. Enter your Blogger URL (e.g., https://yourblog.blogspot.com)
2. Click "Parse" to generate the RSS feed URL
3. Click "Run Scraper" to fetch all posts
4. XML files will download automatically to your Downloads folder
5. View the summary of posts per year

## Deployment

```bash
npm run deploy
```
